# Электронный журнал плавки

## Запуск приложения

Есть несколько способов запуска приложения:

### 1. Использование BAT-файла (Рекомендуется для Windows)

Просто дважды щелкните на файле `run_journal.bat`. Это:
- Проверит наличие Python и установит необходимые пакеты
- Запустит приложение в вашем браузере
- Закройте окно командной строки, чтобы завершить работу приложения

### 2. Создание и использование EXE-файла

1. Дважды щелкните на файле `create_exe.bat`
2. Дождитесь завершения процесса создания EXE-файла
3. После успешного создания, запустите файл `dist/Журнал_плавки.exe`
4. Приложение откроется в браузере

### 3. Запуск напрямую через командную строку

```
streamlit run streamlit_app.py
```

## Требования

- Python 3.7 или выше
- Пакеты Python: streamlit, pandas, plotly, openpyxl, xlsxwriter

## Функции приложения

- Просмотр и фильтрация записей журнала плавки
- Добавление новых записей
- Экспорт данных в Excel
- Анализ статистики
- Импорт данных из текстовых сообщений

## Файлы базы данных

- `plavka.db` - База данных SQLite
- `plavka.xlsx` - Резервная копия данных в формате Excel
